What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents an HTML document as a tree of objects. It allows developers to access, manipulate, and modify the structure, content, and style of a webpage using JavaScript.

How do you select an element from the DOM?

To select an element from the DOM, JavaScript provides several methods, depending on how you want to target the element. Here are the most common ways:

1. By ID: Use "document.getElementById("id")" to select an element by its unique id attribute.
2. By Class: Use "document.getElementsByClassName("className")" to select all elements with a specific class. This returns a live HTMLCollection.
3. By Tag Name: Use "document.getElementsByTagName("tagName") "to select all elements of a particular HTML tag (e.g., div, p, h1). This also returns an HTMLCollection.
4. By CSS Selector: Use "document.querySelector("selector")" to select the first element that matches a CSS selector.
*Use "document.querySelectorAll("selector")" to select all elements that match a CSS selector. This returns a static NodeList.

These methods give you the ability to target and manipulate elements within the DOM.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method where a single event listener is added to a parent element to handle events triggered by its child elements, using event bubbling.

Benefits:
1. Efficiency: Fewer event listeners improve performance.
2. Handles dynamic elements: Works on elements added later.
3. Simplified code: One listener for many elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles in the DOM using:

1. Attributes:
Get: element.getAttribute("attributeName");
Set: element.setAttribute("attributeName", "value");
Remove: element.removeAttribute("attributeName");

2. Styles:
Direct style change: element.style.property = "value";
Example: element.style.color = "red";
CSS classes:
Add: element.classList.add("className");
Remove: element.classList.remove("className");
These methods allow dynamic updates to an element's attributes and styles.
